Show: Se podría decir que, se define una función "show" que toma como
argumento un valor de un tipo determinado y lo convierte en una cadena
de caracteres.

 Enum: La clase de tipo "Enum" permite que los tipos que forman una
secuencia puedan ser enumerados. Esta clase de tipo define las funciones
"succ" y "pred", que permiten obtener el siguiente o el anterior valor
de un tipo determinado en la secuencia.

Ota cosa es que la clase de tipo "Enum" también define las funciones "toEnum" y 
"fromEnum", que permiten convertir valores enteros en valores del tipo
enumerado y viceversa.

Bounded: Esta clase de tipo es usada para tipos que tienen un límite 
superior e inferior definido. Esta clase de tipo define las constantes "minBound"
y "maxBound", que representan el valor mínimo y máximo de un tipo determinado.

Read: Permite que los valores de un tipo puedan ser 
convertidos desde una cadena de caracteres a un valor del tipo determinado.
Esta clase de tipo define una función "read" que toma como argumento una
cadena de caracteres y devuelve un valor del tipo determinado. Tambien 
puede lanzar excepciones si el formato no es valido.
