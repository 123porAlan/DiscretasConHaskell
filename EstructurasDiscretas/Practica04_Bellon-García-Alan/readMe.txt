3. ¿Para qué sirve la palabra reservada deriving?
La palabra reservada deriving en Haskell se utiliza para
indicar al compilador que derive automáticamente las
instancias de ciertas clases de tipos para un nuevo tipo
definido por el usuario. En otras palabras, cuando
definimos un nuevo tipo de datos en Haskell, podemos
utilizar la palabra clave deriving seguida de una lista
de las clases de tipo que deseamos que el compilador
derive automáticamente.
